# Install PHP 8.2 with Required Extensions for Laravel 10

This guide explains how to install **PHP 8.2** along with the required extensions for **Laravel 10**, **MySQL**, and **Redis** on an Ubuntu-based AWS EC2 instance.

## Step 1: Update the System

```bash
sudo apt update
sudo apt upgrade -y
```

---

## Step 2: Install Required Package

```bash
sudo apt install -y software-properties-common
```

---

## Step 3: Add the PHP Repository

```bash
sudo add-apt-repository ppa:ondrej/php -y
sudo apt update
```

---

## Step 4: Install PHP 8.2 and Required Extensions

```bash
sudo apt install -y \
php8.2 \
php8.2-cli \
php8.2-common \
php8.2-fpm \
php8.2-mysql \
php8.2-mbstring \
php8.2-xml \
php8.2-bcmath \
php8.2-curl \
php8.2-zip \
php8.2-gd \
php8.2-intl \
php8.2-soap \
php8.2-readline \
php8.2-opcache \
php8.2-redis \
php8.2-imagick
```

---

## Step 5: Verify the Installation

Check the installed PHP version.

```bash
php -v
```

Example output:

```text
PHP 8.2.x (cli)
```

---

## Step 6: Verify Installed Extensions

List all installed PHP modules.

```bash
php -m
```

Or verify specific extensions.

```bash
php -m | grep -E "bcmath|curl|gd|intl|mbstring|mysqli|mysqlnd|openssl|pdo_mysql|redis|soap|xml|zip"
```

---

## Installed Extensions

| Extension | Purpose |
|-----------|---------|
| `php8.2-cli` | PHP Command Line Interface |
| `php8.2-common` | Common PHP files |
| `php8.2-fpm` | PHP FastCGI Process Manager (for Nginx) |
| `php8.2-mysql` | MySQL/MariaDB support |
| `php8.2-mbstring` | Multibyte string support |
| `php8.2-xml` | XML processing |
| `php8.2-bcmath` | Arbitrary precision mathematics |
| `php8.2-curl` | HTTP requests |
| `php8.2-zip` | ZIP archive support |
| `php8.2-gd` | Image processing |
| `php8.2-intl` | Internationalization |
| `php8.2-soap` | SOAP web services |
| `php8.2-readline` | Interactive CLI support |
| `php8.2-opcache` | PHP performance optimization |
| `php8.2-redis` | Redis support |
| `php8.2-imagick` | ImageMagick integration |

---

## Restart PHP-FPM

If you're using Nginx, restart PHP-FPM after installation.

```bash
sudo systemctl restart php8.2-fpm
```
