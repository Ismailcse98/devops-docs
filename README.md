# Learning in Public

I created this repository to document my hands-on learning journey in **Laravel**, **DevOps**, **AWS**, and **Cloud Infrastructure**.

The goal is to keep practical, reusable guides that I can refer back to in future projects while also helping other developers who are learning the same technologies.

This repository contains step-by-step tutorials, configurations, troubleshooting notes, and best practices based on real implementation rather than theory.

## Why this repository?

* Document everything I learn.
* Reuse configurations in future projects.
* Share knowledge with the developer community.
* Help beginners learn from practical examples.
* Build a collection of production-inspired DevOps notes.
